The Render System

The render system is a self contained WebGL renderer. It has the following classes.

  • Mesh - The vertices and primitives of an object.
  • Shader - The GLSL code of an object.
  • Texture - The image applied to an object.
  • Uniforms - The specific values and textures to apply to an object.
  • Model - An object, combining a mesh, a shader, and uniforms.
  • Scene - A set of models to be rendered.
  • Stage - A view of a scene.

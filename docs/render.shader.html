Birch Documentation

Shader

Introduction

The shader contains the GLSL code that renders the object.

A shader is created and destroyed by:

const shader = birch.render.shaders.create();

birch.render.shaders.destroy(shader);

Setting up the Shader

There are two ways to setup the shader. The first is via:

shader.setCodeAndAttributes(vertexCode, fragmentCode, { position: 0, uv: 1, color: 2 });

The vertex and fragment code are simple strings that contain the shader code. The attribute locations are the same ones used in Mesh.setVertexFormat.

The other way to setup the shader is by using a Shader.Options object:

shader.setFromOptions({ attributeLocations: { position: 0, uv: 1, color: 2 }, vertex: vertexCode, fragment: fragmentCode });

The difference is here that you can put the above in a JSON file like so (using the Downloader):

downloader.getJson('shader.json').then((json) => { shader.setFromOptions(json); });
